## 安装parcel
```
  npm install -D parcel-bundler -v1.9.2
  //在运行的时候
  ./node_modules/.bin/parcel index.html
  npx parcel index.html 上面书写形式的简略版
  npx parcel index.html --no-cache 运行的时候不会再有缓存
```
